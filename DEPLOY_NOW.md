# 🚀 DEPLOY YOUR PORTFOLIO TO VERCEL (5 MINUTES)

## METHOD 1: Drag & Drop (EASIEST - NO INSTALLATION!)

### Step 1: Go to Vercel
1. Open your browser
2. Go to: **https://vercel.com/new**
3. Sign up with email or GitHub (it's FREE)

### Step 2: Drag & Drop Your Folder
1. On the Vercel page, you'll see "To deploy a new Project, import an existing Git Repository or deploy from a Template"
2. Scroll down and you'll see: **"Or, deploy a new Project from a local directory"**
3. Click **"Browse"** or drag your entire project folder
4. Wait for upload (might take 1-2 minutes)
5. Click **"Deploy"**
6. DONE! 🎉

Your site will be live at: `https://your-project-name.vercel.app`

## METHOD 2: Using GitHub (Recommended for Updates)

### Step 1: Create GitHub Account
1. Go to **github.com**
2. Sign up (FREE)
3. Verify your email

### Step 2: Install GitHub Desktop
1. Go to **desktop.github.com**
2. Download and install
3. Sign in with your GitHub account

### Step 3: Upload Your Project
1. Open GitHub Desktop
2. Click **File** → **Add Local Repository**
3. Click **Choose...** and select your project folder: `C:\Users\Tegona Tube\Downloads\estif`
4. Click **"create a repository"** (blue link)
5. Click **"Create Repository"**
6. Click **"Publish repository"** (top right)
7. Uncheck "Keep this code private" if you want it public
8. Click **"Publish repository"**

### Step 4: Deploy to Vercel
1. Go to **vercel.com/new**
2. Sign up/login with GitHub
3. You'll see your repository listed
4. Click **"Import"** next to your repository
5. Click **"Deploy"**
6. Wait 1-2 minutes
7. DONE! 🎉

Your site will be live at: `https://your-project-name.vercel.app`
