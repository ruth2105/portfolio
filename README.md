# Ethiopian Artist Portfolio

A professional portfolio website designed for Ethiopian visual artists, inspired by contemporary Ethiopian artist portfolios.

## Features

- **Fixed Sidebar Navigation**: Professional left sidebar with artist info and navigation
- **Portfolio Gallery**: Showcase artwork with detailed information and lightbox viewing
- **About Section**: Comprehensive artist biography, education, and exhibition history
- **Events Section**: Upcoming exhibitions and events
- **Contact Section**: Multiple contact methods and social media links
- **Responsive Design**: Works on desktop, tablet, and mobile devices
- **Professional Styling**: Clean, modern design that highlights artwork

## Setup Instructions

1. **Replace Placeholder Content**:
   - Update `[Your Name]` with the actual artist name throughout the HTML
   - Replace contact information with real details
   - Fill in education, exhibitions, and biography information

2. **Add Images**:
   Create an `images` folder and add the following images:
   - `bg.jpg` - Background image for desktop (1920x1080px recommended)
   - `bg-mobile.jpg` - Background image for mobile (768x1024px recommended)
   - `artist-photo.jpg` - Professional photo of the artist (400x500px)
   - `artwork1.jpg` through `artwork6.jpg` - Gallery artwork images (800x600px recommended)
   
3. **Add Resume**:
   - Add `resume.pdf` file to the root directory
   - Update the resume link in the About section

4. **Customize Content**:
   - Update artist biography and background
   - Add actual artwork titles, dimensions, and descriptions
   - Update social media links
   - Modify exhibition and education lists

## Customization Tips

- The sidebar is fixed on desktop and becomes a mobile menu on smaller screens
- Portfolio items can be categorized (painting, drawing, screenprint, etc.)
- Background images create the artistic atmosphere
- Color scheme uses green (#18d26e) as the primary accent color
- Typography uses Open Sans, Raleway, and Poppins fonts
